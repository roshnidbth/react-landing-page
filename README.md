# React Landing Page

This is a simple landing page built using React.js.

## Features
- Responsive layout
- Navbar, Hero, Features and Footer sections
- Styled using inline CSS
- Deployed using Netlify

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS
- Netlify (for deployment)


## How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/USERNAME/react-landing-page.git
````

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm start
```

The app will run on:
[http://localhost:3000](http://localhost:3000)

## **Deployment**

The project is deployed using Netlify and is publicly accessible at:
**[https://landing-page-using-reactjs.netlify.app/](https://landing-page-using-reactjs.netlify.app/)**
